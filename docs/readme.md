# Документация по проекту Snowflake

## Описание

Snowflake - это десктоп приложение для управления книгами, сериями книг и авторами. Приложение позволяет создавать и редактировать информацию о книгах, организовывать их в серии, управлять авторами и создавать связи между книгами.

## Основные возможности

1. Управление проектами

- Создание новых проектов
- Открытие существующих проектов
- История недавних проектов
- Настройка проекта
- Экспорт/импорт проектов

2. Управление книгами

Создание новых книг с указанием:

- Названия и описания
- Авторов
- Даты публикации
- Количества страниц
- ISBN
- Жанров
- Статуса (черновик/опубликовано/в архиве)

3. Управление сериями книг
   Создание серий различных типов:

- Дилогия (2 книги)
- Трилогия (3 книги)
- Тетралогия (4 книги)
- Пенталогия (5 книг)
- Гексалогия (6 книг)
- Гепталогия (7 книг)
- Серия (более 7 книг)

4. Связи между книгами

Поддерживаются следующие типы связей:

- Сиквел (продолжение)
- Приквел (предыстория)
- Спин-офф (ответвление)
- Побочная история
- Адаптация
- Ремейк
- Альтернативная версия

5. Дополнительные инструменты

- Синопсис книги
- План книги (главы и сцены)
- Карта персонажей

## Стек технологий

- Vue 3 + TypeScript
- Vue Router для навигации
- Vue I18n для локализации (поддержка EN/RU)
- Naive UI для компонентов интерфейса
- Pinia для управления состоянием
- Rust для бэкенда (desktop)
- Tauri для сборки desktop приложения

## Локализация

- Английский (EN)
- Русский (RU)

Эти изменения отражают добавление нового модуля Project и его интеграцию в существующую архитектуру приложения. Project теперь является корневым модулем, который управляет всеми остальными модулями и их данными.
