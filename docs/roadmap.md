# Дорожная карта развития Snowflake Desktop

## Введение

Данный документ описывает предполагаемый план развития Snowflake Desktop, он может быть пересмотрен в любой момент.

## Возможные улучшения

- **Улучшение текстового редактора**

  - Расширенное форматирование (списки, цитаты, заголовки разных уровней)
  - Интеграция проверки орфографии и грамматики
  - Возможность вставки простых таблиц
  - Статистика текста (количество слов, символов, оценка времени чтения)

- **Оптимизация производительности**

  - Улучшение загрузки больших проектов
  - Оптимизация работы с проектами, содержащими большое количество персонажей
  - Более эффективное управление памятью при работе с визуальными инструментами

- **Улучшение системы экспорта**

  - Расширенные настройки экспорта в DOCX
  - Предварительный просмотр экспорта
  - Экспорт карты персонажей в графические форматы (PNG, SVG)

- **Расширение функций планирования**

  - Трекер задач для отслеживания прогресса работы над книгой
  - Установка целей по количеству слов и дедлайнов
  - Визуализация прогресса и статистика

- **Улучшение карты мира**

  - Добавление слоев на карте
  - Масштабирование частей карты
  - Привязка событий к локациям на карте
  - Отображение перемещений персонажей

- **Обновление системы резервного копирования**

  - Расширенные настройки автоматического резервного копирования
  - Облачное резервное копирование (локальный сервер)
  - Сравнение версий и выборочное восстановление

- **Начальная интеграция с AI**

  - Генерация имен персонажей
  - Помощь в создании описаний персонажей и локаций
  - Предложения по развитию сюжета
  - Анализ структуры текста и рекомендации

- **Система заметок**

  - Создание и организация исследовательских заметок
  - Прикрепление заметок к сценам, персонажам и локациям
  - Быстрый поиск по заметкам
  - Организация заметок в древовидную структуру

- **Расширенные инструменты анализа**

  - Анализ частоты использования слов
  - Отслеживание появления персонажей в сценах
  - Анализ эмоционального тона сцен
  - Визуализация арок персонажей

- **Улучшенные возможности для работы с сериями**

  - Детальное планирование серии книг
  - Общие элементы лора для всей серии
  - Отслеживание развития персонажей через несколько книг
  - Управление сюжетными линиями, охватывающими несколько книг

- **Базовые возможности совместной работы**

  - Экспорт и импорт отдельных компонентов проекта
  - Система комментариев для редакторов и соавторов
  - Экспорт специальных версий для редакторов
  - Сравнение и объединение изменений

- **Расширенная система тегов**

  - Теги для всех элементов проекта
  - Фильтрация по тегам
  - Цветовая маркировка тегов
  - Автоматические тематические теги

- **Интеграция с внешними сервисами**

  - Синхронизация с облачными хранилищами (Google Drive, Dropbox)
  - Интеграция с Grammarly для расширенной проверки грамматики
  - Публикация в форматах для электронных книг
  - Экспорт для платформ самиздата

- **Улучшенный пользовательский интерфейс**

  - Полностью настраиваемый интерфейс
  - Новые темы оформления
  - Режим фокусировки для письма
  - Улучшенная доступность

- **Мультимедийная поддержка**

  - Расширенная поддержка изображений
  - Добавление аудио заметок
  - Встроенные ссылки на исследовательские материалы
  - Галерея референсов для персонажей и локаций

- **Полный набор функций для профессионального использования**

  - Стабильность и оптимизация всех компонентов
  - Полноценная документация
  - Пакет шаблонов для различных жанров
  - Экосистема дополнений

- **Расширенное управление проектами**

  - Портфолио проектов
  - Шаблоны для быстрого создания новых проектов
  - Импорт данных из других программ для писателей
  - Единый интерфейс для управления несколькими проектами

- **Полноценная система совместной работы**

  - Работа над проектом в реальном времени
  - Управление ролями и разрешениями
  - История изменений с указанием авторов
  - Чат и обсуждения внутри проекта

- **Расширенная AI-интеграция**

  - Анализ целостности сюжета
  - Поиск нестыковок в тексте и лоре
  - Продвинутые предложения по развитию персонажей
  - Помощь в генерации описаний мира

- **Web-версия приложения**

  - Синхронизация между десктопной и веб-версией
  - Адаптивный дизайн для мобильных устройств
  - Базовые функции для работы в офлайн-режиме
  - Совместимость с основными браузерами

- **Marketplace для дополнений и шаблонов**

  - Экосистема пользовательских плагинов
  - Платные и бесплатные шаблоны для различных жанров
  - Возможность для сообщества создавать и делиться расширениями
  - Интеграция с платформами для продажи контента

- **Интеграция с издательскими платформами**
  - Прямой экспорт в форматы основных издательств
  - Автоматическая подготовка рукописи согласно требованиям
  - Интеграция с платформами самиздата
  - Инструменты для маркетинга и продвижения книги

## Экспериментальные функции

Следующие функции находятся в стадии активного исследования и тестирования. Они могут появиться в будущих версиях приложения, но конкретные сроки пока не определены.

### AI-ассистент для писателей

- **Статус**: Концептуальное тестирование
- **Описание**: Встроенный ИИ-помощник для генерации идей, преодоления творческого блока и улучшения текста.
- **Текущие результаты**:
  - Успешные эксперименты с генерацией имен и базовых описаний
  - Проблемы с последовательностью при генерации больших объемов текста
  - Хорошие результаты при редактировании и улучшении существующего текста\

### Голосовой ввод и диктовка

- **Статус**: Исследование технологий
- **Описание**: Система распознавания речи для диктовки текста и голосового управления приложением.
- **Текущие результаты**:
  - Хорошие результаты с использованием сторонних API
  - Трудности с распознаванием специфической терминологии
  - Высокая точность при использовании для команд

### Система моделирования персонажей

- **Статус**: Концептуальная разработка
- **Описание**: Продвинутая система для создания психологически достоверных персонажей с моделированием их поведения в различных ситуациях.
- **Текущие результаты**:
  - Разработана базовая модель психологических профилей
  - Интересные результаты в экспериментах с симуляцией диалогов
  - Необходима доработка интерфейса для удобного использования

### Параллельное редактирование в реальном времени

- **Статус**: Активная разработка
- **Описание**: Технология, позволяющая нескольким авторам одновременно работать над одним документом с мгновенной синхронизацией изменений.
- **Текущие результаты**:
  - Рабочий прототип на базе технологии CRDT
  - Хорошая производительность при небольшом количестве пользователей
  - Проблемы с решением конфликтов при сложном форматировании
